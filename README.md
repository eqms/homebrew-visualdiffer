# Homebrew Tap for VisualDiffer

Homebrew Cask for installing [VisualDiffer](https://github.com/eqms/visualdiffer) — a visual folder and file comparison tool for macOS.

## Installation

```bash
brew install --cask eqms/visualdiffer/visualdiffer
```

## Update

```bash
brew upgrade --cask visualdiffer
```

## Uninstall

```bash
brew uninstall --cask visualdiffer
```

## Requirements

- macOS 13.5 (Ventura) or later
- Homebrew

## What gets installed

- **VisualDiffer.app** in `/Applications/`
- **visdiff** CLI tool (symlinked from the app bundle)
